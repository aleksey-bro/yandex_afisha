# yandex_afisha
# Анализ пользовательского поведения Yandex Афиша

## Описание
Проект по анализу факторов, влияющих на повторные покупки пользователей.

## Данные
- 290 611 транзакций
- 21 309 пользователей

## Основные результаты
- Высокая доля пользователей с 1 заказом
- Интервал между заказами — ключевой фактор удержания
- Поведенческие признаки важнее статических

## Запуск проекта

1. Создать виртуальное окружение
2. Установить зависимости:
   pip install -r requirements.txt
3. Запустить ноутбук:
   jupyter notebook
